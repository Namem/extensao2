2b — deploy tflite micro
AI

Integrar modelo Edge Impulse ao firmware

Após exportar do Edge Impulse como Arduino Library

O modelo foi exportado do Edge Impulse como Arduino
Library (.zip). Integre ao firmware:

1. Descompacte em firmware/esp32s3_ceres/lib/ei_ceres/
2. Adicione lib_deps no platformio.ini apontando para
   esta pasta local

3. Crie src/inference.h e src/inference.cpp:
   struct InferenceResult {
     uint8_t classe_id;
     const char* nome_classe;
     float confianca;
     uint32_t latencia_ms;
   };
   InferenceResult run_inference(camera_fb_t* frame);

4. run_inference() deve:
   - Redimensionar frame para resolução do modelo
     (96x96 ou 160x160 conforme exportado)
   - Normalizar pixels para [-1, 1]
   - Alocar tensor arena na PSRAM (MALLOC_CAP_SPIRAM)
   - Medir tempo com esp_timer_get_time()
   - Retornar classe com maior probabilidade

5. main.cpp: captura frame, chama run_inference(),
   imprime no serial:
   INFERENCIA: [classe] conf=[0.87] lat=[145ms]
   RAM_livre=[6234KB]

Mínimo esperado: latência < 300ms, RAM livre > 4MB
